# Contact-Keeper
> Simple MERN app
## Usage
Create "config/default.json & config/production.json" and update mongoURI & jwtSecret with your own values.
```
{
  "mongoURI": "<value>"
  "jwtSecret": "<value>"
}
```
## Install Dependencies
```
npm install
```
## Run App
```
npm run dev
```
- Version 1.0.0
